Este projeto realiza detecção automática de fogo, fumaça e faíscas através de palavras chaves utilizando o modelo BLIP (Image Captioning) combinado com análise semântica.
Também envia alertas em tempo real via:

- E-mail;
- Som e alerta por voz;
- Blynk IoT (aplicativo e web);
  

Além disso, registra:

 - Fotos dos eventos detectados com legendas geradas pela IA;
 - Log de eventos em CSV;
 - Tempo de processamento de cada imagem;
